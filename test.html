Why Griot

Complete, not complicated.

A real editing experience with multimedia, inline formatting, and a full default CSS — zero runtime dependencies.

🧱

19 Block Types

Text, callouts (4 variants), code, lists, image, video, audio, gallery, embed, table, divider, timeline ref, book citation.

🎬

Rich Multimedia

Auto-detects YouTube, Vimeo, Spotify, and SoundCloud URLs. Falls back to native audio/video. Gallery in grid, masonry, carousel, or strip.

✍️

12 Inline Tokens

Bold, italic, underline, strike, code, highlight, color marks, links, inline images, and event/cite chips — all with a standalone parser.

⌨️

Markdown Shortcuts

Type # , > , - , ``` etc. to convert a block instantly as you type.

🎨

Ships with CSS

A complete dark theme in griot.css — all values are CSS variables so you can retheme everything with a few overrides.

🔀

Immutable Ops

Every document mutation returns a new object. Undo/redo, diffing, and syncing are trivial. Up to 200 history snapshots.

🔲

Floating Toolbar

Select any text to reveal bold, italic, underline, strikethrough, code, highlight, link insertion, and color marking.

⚡

Slash Commands

Type / in any empty block to open a searchable, keyboard-navigable palette grouped by category.

👁️

Live Preview

Every inline-capable text block shows a rendered preview strip beneath the plain-text input as you type.

Inline Markup

12 inline tokens, one lightweight parser.

The inline lexer is fully independent — use renderInlineToDOM() or renderInlineToHTML() anywhere.

**bold**→bold
*italic*→italic
__underline__→underline
~~strikethrough~~→strikethrough
`inline code`→inline code
==highlight==→highlighted
{#6366f1:colored text}→colored text
[label](url)→label
[[event:id|label]]→
[[cite:id|label]]→

API Reference

Clean, composable exports.

Every module is independently importable. The inline parser, history stack, and document operations work without the editor.

// Editor — full editing lifecycle import { Editor, createDocument } from 'griot'; const editor = new Editor(container, { doc: createDocument([ { id: 'b1', type: 'heading', text: 'Hello', meta: { level: 1 } }, { id: 'b2', type: 'paragraph', text: 'Start writing…' }, ]), books: [], // for book_citation blocks onChange: (doc) => saveToServer(doc), // debounced 400ms onEventClick: (eventId) => openTimeline(eventId), onCiteClick: (blockId) => scrollToBlock(blockId), onRequestBookPicker: (blockId, cb) => { // open your own picker UI, then call cb with selection cb({ bookId: 'book1', unitId: 'ch3', quote: '', note: '' }); }, }); editor.focus('b1'); // focus a block editor.setDoc(newDoc); // replace document editor.destroy(); // clean up
// Viewer — read-only, same blocks import { Viewer } from 'griot'; const viewer = new Viewer(container, { doc: myDocument, books: myBooks, highlightBlockId: 'b2', // initial highlight onEventClick: (id) => openEvent(id), onCiteClick: (id) => openCitation(id), }); // Scroll to a block + 2.2s CSS pulse animation viewer.setHighlight('b1'); viewer.setDoc(updatedDoc); // live update viewer.destroy();
// Inline parser — fully standalone import { tokenizeInline, renderInlineToDOM, renderInlineToHTML, TOKEN } from 'griot'; const text = 'This is **bold**, `code`, ==highlight==, {#6366f1:colored},' + ' and a [[cite:b1|citation chip]].'; // To DOM (for browser use) const frag = renderInlineToDOM(text, { onEventClick: (id) => console.log('event:', id), onCiteClick: (id) => console.log('cite:', id), }); el.appendChild(frag); // To HTML string (for SSR / export) const html = renderInlineToHTML(text); // Raw tokens const tokens = tokenizeInline(text); // → [{ type: TOKEN.TEXT }, { type: TOKEN.BOLD, text: 'bold' }, …]
// Immutable document operations import { createDocument, createBlock, updateBlock, insertBlockAfter, removeBlock, splitBlock, mergeBlockWithPrev, moveBlock, } from 'griot'; // Every operation returns a NEW document let doc = createDocument(); doc = updateBlock(doc, 'b1', { text: 'Updated text' }); doc = updateBlock(doc, 'b1', { meta: { level: 3 } }); // meta is shallow-merged const nb = createBlock('paragraph', { text: 'New block' }); doc = insertBlockAfter(doc, 'b1', nb); // splitBlock returns [newDoc, newBlockId] const [doc2, newId] = splitBlock(doc, 'b1', 12); // mergeBlockWithPrev returns [newDoc, prevId, mergeOffset] const [doc3, prevId, offset] = mergeBlockWithPrev(doc2, newId);
// History — linear undo/redo stack, max 200 snapshots import { History } from 'griot'; const history = new History(initialDoc); history.push(newDoc); // new undo point, discards redo future history.replace(doc); // overwrite current — no new undo point // (used during fast typing, debounced 400ms) history.undo(); // returns previous document history.redo(); // returns next document history.current; // current document history.canUndo(); // boolean history.canRedo(); // boolean
