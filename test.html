Why Griot

Everything you need.
Nothing you don't.

Built for developers who want a real editing experience without the bloat of a framework-heavy dependency.

🧱

Block-based

Every element is a block — headings, paragraphs, callouts, code, media, tables, dividers, citations, and timelines.

✍️

Inline Markup

Lightweight syntax for bold, italic, highlights, colored text, inline code, links, and custom chip types.

⚡

Zero Dependencies

Pure ES modules. No React, no Vue, no bundler required. Drop it in via CDN or install from npm.

🔀

Immutable Ops

Every document operation returns a new document object, making undo/redo, diffing, and syncing trivial.

🎨

Unstyled by Design

No opinionated CSS. All elements use semantic griot- class names. Style it your way.

🔧

Schema-Driven

Block types live in a single schema file. Add new types at runtime or extend the schema statically.

📖

Dual Mode

The same document renders in both Editor and read-only Viewer mode — same blocks, same inline tokens.

⌨️

Slash Commands

Type / anywhere to open the block picker. Fully keyboard-navigable.

↩️

Undo / Redo

Built-in linear history stack. Use it via the Editor or independently in your own app.

Block Types

16+ block types out of the box.

From simple text to rich media, every block is defined in a single extensible schema.

Inline Markup

Rich text, lightweight syntax.

The inline parser is independent — use it standalone anywhere you need formatted text rendering.

**bold** → bold
*italic* → italic
~~strikethrough~~ → strikethrough
`inline code` → inline code
==highlight== → highlighted text
{#c8f060:colored} → colored
[[cite:b1|Citation]] → 📎 Citation
[[event:e1|Event]] → ⚡ Event
API Reference

Clean, composable API.

Immutable document operations, standalone inline rendering, and a simple event model.

// Initialize the editor import { Editor, createDocument } from 'griot'; const doc = createDocument([ { id: 'b1', type: 'heading', text: 'My Document', meta: { level: 1 } }, { id: 'b2', type: 'paragraph', text: 'Start writing...' }, ]); const editor = new Editor(container, { doc, onChange: (newDoc) => console.log('changed', newDoc), onEventClick: (id) => console.log('event:', id), onCiteClick: (id) => console.log('cite:', id), });
// Read-only viewer import { Viewer } from 'griot'; const viewer = new Viewer(container, { doc: myDocument, books: myBooks, highlightBlockId: 'b2', onCiteClick: (id) => showCitation(id), }); // Scroll to and highlight a block viewer.setHighlight('b1');
// Use the inline renderer standalone import { renderInlineToDOM, renderInlineToHTML } from 'griot'; const text = 'This is **bold** and `code` with ==highlight=='; // Render to DOM fragment const frag = renderInlineToDOM(text, { onEventClick: (id) => console.log(id), onCiteClick: (id) => console.log(id), }); el.appendChild(frag); // Or render to an HTML string const html = renderInlineToHTML(text);
// Immutable document operations import { updateBlock, insertBlockAfter, removeBlock, splitBlock } from 'griot'; // Every operation returns a NEW document let doc = createDocument(); doc = updateBlock(doc, 'b1', { text: 'Updated!' }); doc = insertBlockAfter(doc, 'b1', createBlock('paragraph')); doc = splitBlock(doc, 'b2', 12); // split at offset doc = removeBlock(doc, 'b3');
